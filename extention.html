Google Chrome Extension

Google chrome Extention for assisting in all aspects of dyslexia from visibility, reading, writing and spelling assistance.

Dyslexify

Dyslexify is a google chrome extention made for assisting people who may or may not have dylexia and the aim is to help the individuals learning difficulty by providing assistance need the following;

  • Visibility
  • Reading
  • Spelling
  • Writing

    • The visibility aspect of the product is allowing the user to set a preference of the background and text colour for both the google extension and other websites within google chrome. This is because some people with dyslexia find that coloured overlays help them as coloured filters and lenses can alleviate visual distortions for people with dyslexia.

    The reading aspect of the product is allowing the user to use the built-in text to speech (TTS) which will be powered by googles TTS API (application programming interface). The product will allow the user to manually input words/sentences into the extensionâ€™s popup for text to speech as well as the user will be able to input highlighted/selected text on the browser onto the text to speech.

    The spelling aspect of the product is allowing the user to play a game within the extension which will help them slowly develop there spelling skills/knowledge by repeatedly having random words with an arrange of various levels from them to use depending on the intensity of the dyslexia.

    The writing aspect of the product is allowing the user to use the built-in speech to text (STT), which allows the user to speak into their microphone and then the extension will translate the verbal words spoken into the text field allowing them to use it within websites rather than having to struggle writing and spelling the words until the spelling game helps them improve.
